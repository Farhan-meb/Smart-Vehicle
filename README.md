# Smart-Vehicle
About Project : A vehicle that will go from a given source to a given destination through a path. There might be a lot of paths from the source to destination but the vehicle will always use the most benefitted path. User can draw the map the way they want. If the vehicle finds any obstacles (traffic jam , broken road, etc), it will simply avoid the road and will search for different road. e.t.c 5.
